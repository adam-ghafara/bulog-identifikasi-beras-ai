Tentang

Web Prototipe ini adalah pengembangan awal (Prototipe) dari aplikasi Identifikasi Beras berbasis AI yang akan dirancang. Aplikasi ini bertujuan untuk membantu masyarakat serta instansi dalam mengidentifikasi kualitas komoditi beras yang dimiliki. Dengan menggunakan teknologi AI, aplikasi ini dapat mengidentifikasi kualitas beras berdasarkan gambar yang diunggah oleh pengguna.

Web ini dibuatkan dalam rangka untuk memenuhi prosedur kegiatan magang di PT Perum Bulog.

Web ini tidak keterkaitan erat dengan PT Perum Bulog dan hanya digunakan sebagai penelitian.

Komponen Web:

  • AI: Dibuatkan menggunakan Tensorflow Keras yang dikembangkan menggunakan algoritma Convolutional Neural Network (CNN).
  • Web: Dibuatkan menggunakan html dasar, dan dijalankan menggunakan Python Flask.
